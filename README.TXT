------------------------------------------------------------------------
Ihr könnt abwechselnd Katzen ziehen, bis einer von euch eine Detonating Cat zieht.
Sofern ihr nicht im Besitz einer Entschärfungskatze seid, explodiert ihr umgehend, sterbt und scheidet aus.
Die Entschärfungskatze kann die Detonating Cat beruhigen und vom Explodieren abhalten, zum Beispiel mit einem Laserpoiner, Katzenminze oder Bauchkraulen.
Die übrigen Karten dienen nur einem Ziel: Sie unterstützen euch dabei, eine Detonating Cat zu verhindern.
